## Comment ça marche

Chaque note est un simple fichier Markdown placé dans le dossier `content/`.
Les liens entre notes s'écrivent avec la syntaxe `[[wikilink]]`, et l'arborescence
des dossiers se reflète automatiquement dans l'explorateur à gauche.

> [!tip] Astuce
> Crée un nouveau dossier dans `content/` pour ouvrir une nouvelle section. La page
> d'index de chaque section est générée automatiquement si aucun `index.md` n'existe.
